# RansomwareNodeJs

<p align="center">Projeto desenvolvido para o curso Criação de <strong>Ransomware<strong> multiplataforma com JavaScript</p>

## Instalação
Para iniciar o servidor execute:
```bash
cd server
yarn install
```
Para iniciar o client execute:
```bash
cd client
yarn install
```
